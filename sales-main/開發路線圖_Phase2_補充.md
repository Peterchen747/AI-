#### 📐 規則引擎設計（8 個維度）

| 維度         | 計算方式                             | 閾值設計                                          |
|--------------|--------------------------------------|---------------------------------------------------|
| 毛利率       | `profit / revenue × 100`            | ≥50% 優秀 / 30–50% 健康 / 10–30% 注意 / <10% 危險 |
| 淨利率       | `netProfit / revenue × 100`          | ≥20% 優秀 / 5–20% 健康 / 0–5% 警示 / <0% 危險     |
| 間接費用率   | `weeklyCostsTotal / revenue × 100`   | <25% 正常 / 25–40% 偏高 / >40% 嚴重               |
| 廣告費率     | `adCost / revenue × 100`            | <15% 正常 / 15–30% 偏高 / >30% 危險               |
| 直接成本率   | `cost / revenue × 100`              | 顯示進貨成本佔收入比例（資訊型）                   |
| 月環比趨勢   | `(curr − prev) / prev × 100`        | >+20% 成長 / −10～+20% 穩定 / <−10% 衰退          |
| 類別集中度   | 最大類別營收 / 總營收               | <40% 分散 / 40–60% 中等 / >60% 高風險             |
| 交易量趨勢   | 本月筆數 vs 上月                    | 業務活躍度輔助指標                                 |

**行動建議優先級規則（組合判斷）：**
- `urgent`：淨虧 + 廣告費 >30%、淨虧 + 間接費用 >40%、毛利率 <10%
- `important`：類別集中 >60%、業績下滑 >10%、廣告費 15–30%、毛利率 10–30%
- `optional`：有高毛利品類可加碼、整體健康可持續優化

#### ✅ 驗證方式
1. 開啟儀表板，確認 BarChart 有提示文字「點擊月份查看財務分析 →」
2. 點擊任一月份 bar → 跳轉 `/dashboard/analysis?month=YYYY-MM`
3. 確認毛利率、淨利率數字與 KPI 卡片一致
4. 確認 badge 顏色符合閾值（危險=紅、警示=橘、健康=藍、優秀=綠）
5. 費用分解比例條視覺加總 ≈ 100%
6. 行動建議按 urgent > important > optional 排序
7. 返回儀表板按鈕正常運作
8. `npx tsc --noEmit` 無錯誤
